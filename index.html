Discord Meet API

A Node application based on express mongodb & jsonwebtoken tecnologies.
Jsonwebtoken auth system used as authmiddleware after login.
Tecnologies : express, nodejs, monogodb, mongoose, jsonwebtoken, bcrypt
Security: cors, xss-clean, helmet, rate-limite

User Schema

{firstName: "Jafran",secondName:"Zakaria",email:"jafranbinzakaria@gmail.com",password:'Hello123'(All fields required & password auto encryption enable.)}

Meet(Discord) Schema

{owner:UserObjectId,meetName:'Coding Channel',meetPassword:'meet123',members:[],messgeList:{message;'Hello World',messageSender:UserObjectId,messageTime:defalut set}}

Routing URL

Register User(POST): "https://discord-system-jaifan.herokuapp.com/api/v1/auth/register" post(userSchema)

Login User(POST): "https://discord-system-jaifan.herokuapp.com/api/v1/auth/login" post(email,password)

After Login, Set Jsonwebtoken in header.authorization: "Bearer TOKEN" : then,

Create Meet(POST)(GET): "https://discord-system-jaifan.herokuapp.com/api/v1/meetroom/" post(meetName,meetPassword)get(meet list where you members )

Meet AddMember(POST): "https://discord-system-jaifan.herokuapp.com/api/v1/meetroom/addMember" post(meetName,meetPassword) Meet owner provide those info

Get Meet ID (GET): "https://discord-system-jaifan.herokuapp.com/api/v1/meetroom/:meetid" (if you a member of meet)

Left from Meet(GET): "https://discord-system-jaifan.herokuapp.com/api/v1/meetroom/:meetid/removeMember" (to left the room)

Kickout By Owner(GET): "https://discord-system-jaifan.herokuapp.com/api/v1/meetroom/:meetid/removeMember/:userid" (Owner forcely remove the user)

Send Message to Meet(POST): "https://discord-system-jaifan.herokuapp.com/api/v1/meetroom/:meetid/sendMessage" post(message) (after the meet member)

Remove Message to Meet(GET): "https://discord-system-jaifan.herokuapp.com/api/v1/meetroom/:meetid/removeMember/:msgid" (specify the messageId to remove the message)
